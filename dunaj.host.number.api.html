Coercions to host number types.

bigdec

Available since version 1.0 (alias of clojure.core/bigdec)

not referred automatically

Usage:

  • (bigdec x)

Type signature:

  • (Any) → java.math.BigDecimal

Coerce to BigDecimal

See also: double, float, dunaj.math/Decimal
