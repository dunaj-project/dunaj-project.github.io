ICanonical

Available since version 1.0 (view source)

not referred automatically

PROTOCOL with method -canonical

A value protocol for objects other than strings which however have a canonical textual representation. Types satisfying ICanonical protocol must accept their canonical form in their constructor.

Note that while most such types also return canonical string with →str, it is not always so, see e.g. instants.

See also: canonical?, canonical

-canonical

Available since version 1.0 (view source)

not referred automatically

Usage:

  • (-canonical this)

Type signature:

  • (Any) → java.lang.String

Returns canonical string representation of this.

See also: ICanonical
